# Chaos Game 2D

Visualization of fractal attractors using the Chaos Game algorithm.

## Features
- real-time fractal generation
- Java Swing GUI
- customizable presets

## Technologies
- Java
- Swing
- OOP

## Run

Download repository and run:

Windows:
run.bat

Linux / Mac:
./run.sh
